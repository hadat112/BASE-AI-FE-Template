# Cursor Prompts

This directory contains prompt templates for use with Cursor AI. These templates help structure your requests to the AI assistant, making it easier to get consistent and helpful responses.

## Available Templates

- **component.md**: Template for creating React components
- **page.md**: Template for creating Next.js pages
- **feature.md**: Template for implementing features
- **api.md**: Template for creating API services
- **hook.md**: Template for creating custom React hooks
- **refactor.md**: Template for code refactoring requests
- **debug.md**: Template for debugging issues

## How to Use

1. Copy the content of the appropriate template
2. Fill in the sections with your specific requirements
3. Paste the filled template into a conversation with Cursor AI
4. The AI will respond with code and explanations based on your specifications

## Customizing Templates

Feel free to modify these templates or create new ones to better suit your project needs. The goal is to provide clear instructions to the AI assistant to get the most helpful responses.

## Best Practices

- Be specific in your requirements
- Provide context about how the code will be used
- Mention any dependencies or constraints
- Specify the expected behavior and edge cases
- Reference existing code patterns when applicable
